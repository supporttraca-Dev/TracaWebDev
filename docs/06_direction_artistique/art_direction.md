# Direction Artistique & Style Graphique

## Concept : "Le Patrimoine dans l'Ombre"
Le style graphique s'inspire de la **Dark Fantasy Historique**. On ne montre pas tout de suite, on laisse l'utilisateur découvrir.

### 🎨 Palette de Couleurs (Art)
- **Contrastes Profonds** : Utilisation de dégradés sombres pour simuler la lumière d'une lampe dans une crypte ou le coucher de soleil sur les ruines.
- **Détails Dorés** : Rappel du faste des civilisations passées (Numides, Ottomans).

## Guidelines Assets (Médias)

### 📸 Images & Photographies
- **Style** : Désaturé, contraste élevé, focus sur la texture de la pierre.
- **Format** : `.webp` obligatoire (pour la performance).
- **Nomenclature** : `lieu-type-nom.webp` (ex: `tipaza-archeo-ruines.webp`).

### 🎥 Vidéos
- Les vidéos d'intro doivent être en boucle, sans texte (le texte est géré en HTML par-dessus).
- Format portrait (`intro-mobile.mp4`) et paysage (`intro.mp4`).

### 🏺 3D & Media Guidelines
- **Modèles Sketchfab** : Toujours vérifier que le mode "Shadows" est activé dans l'embed.
- **Performance** : Limiter les textures à 2K pour le web.
- **Configuration** : Caméra bloquée pour éviter que l'utilisateur ne sorte du décor.
