# Guidelines de Rédaction & Contenu

## Ton et Style Éditorial
Le ton de TRACA doit être **Instruit, Poétique et Institutionnel**. 
- **Instruit** : Les données historiques doivent être précises et sourcées.
- **Poétique** : Le texte doit inviter au voyage, utiliser des adjectifs évocateurs (majestueux, millénaire, énigmatique).
- **Institutionnel** : Le projet doit inspirer confiance aux ministères et partenaires.

## Organisation du Contenu
Le contenu est segmenté par "Lieu" ou par "Chronique".
- **Lieu** : Nom, Wilaya, Époque, Histoire courte (150 mots max pour l'audio).
- **Chronique** : Documentaire long, divisé en chapitres clairs avec images d'archives.

---

# Guidelines pour les nouveaux contenus

Tout nouveau lieu ajouté à la plateforme doit suivre cette fiche type :
1. **Identité** : Nom officiel + Nom vernaculaire (si existant).
2. **Localisation** : Coordonnées GPS précises pour la carte.
3. **Narration** : Script audio prêt pour enregistrement.
4. **Media** : Un modèle 3D Sketchfab et 3 photos de haute qualité (4K).

---

# Langue et Traduction
Le français est la langue de référence actuelle.
- **Arabe** : Doit être la deuxième langue prioritaire (standard littéraire).
- **Anglais** : Langue internationale pour le rayonnement mondial.

*Note : Toujours utiliser le plugin RTL pour MapLibre lors de l'intégration de l'arabe.*
