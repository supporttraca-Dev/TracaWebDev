# Documentation Officielle - Projet TRACA

Bienvenue dans le centre de documentation de TRACA. Ce dossier contient l'intégralité des connaissances techniques, artistiques et stratégiques nécessaires pour la production et le développement de la plateforme.

## Guide pour les nouveaux collaborateurs
1. Commencez par lire la **Vision du Projet** (00).
2. Familiarisez-vous avec les **Principes UX** (02) avant toute modification d'interface.
3. Consultez les **Règles de Développement** (04) avant d'écrire une seule ligne de code.
