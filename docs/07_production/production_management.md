# Gestion du Projet & Production

## Structure de l'Équipe (Rôles)
Pour une production complète, TRACA nécessite les profils suivants :
1. **Lead Producer** : Vision globale, budget, partenariats institutionnels.
2. **Lead Developer** : Architecture front-end et performance 3D.
3. **Art Director** : Identité visuelle, asset management (images, vidéos).
4. **Historical Advisor** : Vérification des faits, rédaction des scripts narratifs.
5. **3D Artist** : Acquisition et nettoyage des modèles photogrammétriques.

## Pipeline de Production (Workflow)
Chaque nouveau site passe par ce tunnel :
1. **Recherche** : Documentation historique et accès au site.
2. **Capture** : Photogrammétrie sur site ou acquisition de modèles existants.
3. **Rédaction** : Écriture de la chronique et du script audio.
4. **Intégration** : Création de la page HTML dans `/lieux` et ajout au GeoJSON.
5. **Validation** : Revue par l'Advisor Historique avant mise en prod.

# Organisation du Travail
- **Outils** : Trello/Jira pour les tâches, GitHub pour le code.
- **Daily** : Point technique de 15 min chaque matin.
- **Sprint** : Cycle de 2 semaines pour l'ajout d'une nouvelle "zone" (ex: "La Casbah").
